# Spring Boot Backend Projects for ECE & EEE Students

## Overview
This repository provides **5 industry-relevant Spring Boot backend projects** specially designed for ECE and EEE students.  
These projects simulate real-time embedded, IoT, automation, and energy monitoring backend systems using REST APIs + PostgreSQL.

## Projects Included

1. **IoT Device Monitoring System**  
   Register IoT devices, update status, monitor activity logs.

2. **Smart Energy Meter Backend**  
   Record electricity usage, generate monthly consumption reports, user billing API.

3. **Sensor Data Logger System**  
   Store temperature, humidity, vibration, or custom sensor readings with timestamps.

4. **Home Automation API**  
   Control devices (fan, light, AC), manage states (ON/OFF), schedule automation tasks.

5. **Solar Power Monitoring System**  
   Track solar panel output, battery levels, efficiency, and power generation logs.

## Tech Stack
- Java 17  
- Spring Boot  
- Spring Web  
- Spring Data JPA  
- PostgreSQL  
- Maven
